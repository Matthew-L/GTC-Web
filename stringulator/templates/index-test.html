Welcome!

Stringulator allows you to create custom guitar string sets. After you build a set, you can save them to your profile to compare to other sets you've tried. Don't want to make your own String Set? Use the search bar to find custom sets other Stringulator users are rocking! If you have questions on how to use the site check out the GitHub Readme or use the Contact Us page to send us a message.
